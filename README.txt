In order to run the program follow the next steps:
1. Make sure all the appropriate software is installed and configured (mysql, django, python).
2. navigate to the main folder of the project (with manage.py)
3. from the command line enter following commands:
	3.1. python manage.py makemigrations
	3.2. python manage.py migrate
	3.3. python manage.py runserver
